# Health Monitoring Streamlit App

This capstone prototype demonstrates:
- sign-in with hashed demo credentials
- CSV ingestion and upload workflow
- preprocessing and cleaning
- Isolation Forest anomaly detection
- rolling risk scoring
- short-term risk forecasting
- personalized profile-based recommendations
- weekly report summaries
- daily summary reporting
- downloadable CSV and text reports
- a refinement comparison panel

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
